# n8n Starter Workflows

These workflow exports are disabled templates imported into the live `n8n` instance.

- `document-intake-webhook.json`: webhook-first document intake skeleton for future Paperless or scanner integrations.
- `admin-review-template.json`: scheduled admin review template for backups, alerts, and inbox checks.
- `monthly-finance-template.json`: scheduled finance review template for budget reconciliation and recurring tasks.

They are intentionally credential-free and disabled by default. Adjust schedules, add credentials, then activate them in the `n8n` UI.
